Drone Delivery Simulation

Made with: HTML, CSS, JavaScript, C++

Docker Image

Developing this drone simulation was one of my favorite projects I have worked on. I collaborated with 3 other teammates to create a drone simulation that virtually delivered items to entities on a map. I thrived on the collaborative process, especially employing complex algorithms to create a visually compelling and functional web interface. We integrated several advanced search algorithms, including A*, Dijkstra, DFS, and BFS, to optimize the drone's navigation routes to the designated entities. A critical aspect of our simulation was considering the drone's battery life. We programmed the drone to automatically reroute to a nearby charging station if its battery level became critically low during a delivery. This feature not only added realism but also involved intricate problem-solving regarding power management. Moreover, we enhanced user interaction with the implementation of real-time notifications. These alerts kept users informed about the drone's status and provided updates on the package delivery process, enhancing the overall user engagement with our simulation.

Dancing Ants Simulation

Made with: HTML, Typescript, JavaScript

Live

The dance simulation utilizes data from the Carnegie mellon motion database to animate computer graphics characters. To animate the characters, skeleton-based mocap data were used to help with smoothly showing movements. Techniques such as scaling, rotating, and translating were used to transform basic geometric 3D shapes into complex scenes and characters. By skillfully manipulating the mocap data, various animation effects could be implemented, including loop animation for seamless motion continuity and overlaying motion clips for character actions.

Earthquake Visualization

Made with: HTML, Typescript, JavaScript

Live

The Earthquake Visualization simulation dynamically displays earthquake events that occurred between 1905 and 2007, offering both 2D map and 3D globe perspectives. This simulation enhances visual detail by applying textures to 3D objects and utilizes vertex buffers to render deforming 3D meshes, facilitating seamless transformations between the 2D and 3D views.

Hole in Ground Game

Made with: HTML, Typescript, JavaScript

Live

The purpose of the 'Hole in the Ground Game' is to develop an engaging 3D physics-based video game, drawing inspiration from the innovative gameplay of the 2018 hit, Donut County. In this game, players manipulate a hole in the ground that enlarges each time an object falls into it. The development process involved crafting 3D geometric objects and managing 3D points, vectors, and geometric transformations to simulate realistic interactions. A key aspect of this game is the balance between realism and gameplay, ensuring that the physics implementation enhances the gaming experience without compromising smoothness or playability.
